# Projeto HTML e CSS

Atividade com duas paginas:

- `index.html`: pagina inicial com 6 imagens.
- `videos.html`: pagina com 2 videos locais.
- `estilo.css`: estilo da pagina inicial.
- `videos.css`: estilo da pagina de videos.
- Pasta `imagens`: imagens usadas no site.
- Pasta `videos`: videos locais usados na pagina de videos.

Para abrir o site, basta clicar duas vezes no arquivo `index.html`.
